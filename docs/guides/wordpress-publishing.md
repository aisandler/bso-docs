### **Step 1: Convert Markdown to WordPress** (15 minutes)

**1.1 Open the markdown file**
- Navigate to the article location
- Open in any text editor

**1.2 Copy article content**
- Copy everything AFTER the meta description line
- Skip the meta description (we'll add it separately in WordPress SEO settings)

**1.3 WordPress converts markdown automatically**
- WordPress Gutenberg editor recognizes markdown formatting
- `##` becomes H2 headings
- `-` becomes bullet lists
- `**bold**` becomes bold text
- Tables remain properly formatted

**1.4 Create new post in WordPress**
- Go to: **Posts ‚Üí Add New**
- Paste the copied content
- WordPress auto-converts markdown to blocks

---

### **Step 2: Configure Post Settings** (10 minutes)

**2.1 Post Title**
- Copy the H1 title from the markdown file
- Example: "What Is a Legal Billing Partnership?"

**2.2 Permalink (URL Slug)**
- WordPress auto-generates from title
- Edit to match: `what-is-legal-billing-partnership`
- Keep it clean, lowercase, hyphenated

**2.3 Category Selection**
- **Primary Category**: Legal Billing Partnerships
- **Secondary**: Add relevant categories based on article tier
  - Tier 1: Legal Billing Partnerships
  - Tier 2: Cash Flow, Operations
  - Tier 3: Technology, Tabs3
  - Tier 4: Thought Leadership, Industry Trends

**2.4 Tags**
Add 5-8 relevant tags:
- Legal billing
- Mid-size law firms
- Billing partnerships
- [Article-specific keywords]

**2.5 Excerpt**
- Copy the meta description from the markdown file
- Paste into excerpt field
- Excerpt appears in search results and social shares

**2.6 Featured Image**
- Upload or select relevant image
- Alt text: Include primary keyword
- Example alt text: "Legal billing partnership services for mid-size law firms"

**2.7 Author**
- Select: Jared McDuffey, Jaclyn McDuffey, or "Beyond Square One" (company author)
- Consider company author for consistency

---

### **Step 3: SEO Optimization** (10 minutes)

**Using Yoast SEO or Rank Math:**

**3.1 Focus Keyword**
- Enter primary target keyword from article
- Example: "legal billing partnership"

**3.2 SEO Title**
- Use article H1 or customize for search
- Keep under 60 characters
- Include keyword near beginning
- Example: "What Is a Legal Billing Partnership? | Beyond Square One"

**3.3 Meta Description**
- Copy from markdown file meta description line
- Keep exactly 155 characters
- Include keyword and CTA
- Example: "Legal billing partnerships combine US-based expertise with ongoing services. Learn how they differ from offshore outsourcing. Beyond Square One guide."

**3.4 Readability Check**
- Yoast/Rank Math will analyze readability
- Ensure green or orange score (not red)
- Articles are already optimized for readability

**3.5 SEO Score**
- Aim for green (good) or orange (okay)
- Articles are pre-optimized for SEO
- Address any critical warnings

## üö® Common Issues & Troubleshooting

### **Issue 1: Markdown Formatting Not Converting**

**Problem:** Tables or formatting look broken after pasting

**Solution:**
- Use WordPress "Code Editor" (not Visual Editor)
- Paste markdown directly
- Switch back to Visual Editor to see converted formatting
- Manual cleanup if needed

---

### **Issue 2: Featured Images Missing Alt Text**

**Problem:** Forgetting alt text reduces SEO value

**Solution:**
- Always add alt text when uploading images
- Include target keyword naturally
- Describe the image content
- Template: "[Keyword] - [Image description]"

---

### **Issue 3: Internal Links Breaking**

**Problem:** Links use wrong format or point to drafts

**Solution:**
- Always use published article URLs
- Test all links after publishing
- Use relative URLs when possible: `/article-slug/`
- Avoid linking to drafts (readers get 404 errors)

---

### **Issue 4: Meta Description Too Long/Short**

**Problem:** Yoast/Rank Math shows red warning

**Solution:**
- Exactly 155 characters (Google's display limit)
- Use character counter tool
- Prioritize keyword + value proposition + CTA
- Trim or expand as needed

---

### **Issue 5: Low SEO Score After Upload**

**Problem:** Yoast/Rank Math shows orange/red SEO score

**Solution:**
- Check focus keyword density (should appear 5-10 times naturally)
- Ensure keyword in: Title, first paragraph, H2 heading, meta description
- Add keyword to alt text (1-2 images)
- Internal/external links present (3-5 each)
- Articles are pre-optimized, so most should score green
